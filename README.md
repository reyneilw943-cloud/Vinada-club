This website represents the official online home of the VINADA Club, a vibrant and culturally rich student organization at the University of the Southern Caribbean. Its purpose is to highlight the identity, activities, and values of the club by providing visitors with a clear and engaging overview of who we are and what we do.
The site features dedicated sections that showcase our diverse cultures, reflecting the unique backgrounds and traditions of our members. It also introduces our executive team, allowing students and visitors to learn more about the individuals responsible for guiding the club’s vision and initiatives.
In addition, the website includes contact pages for inquiries, collaborations, and membership opportunities. It outlines the various ways we contribute to both the university and the wider community through service, cultural expression, and student engagement.
A major focus of the website is to highlight the club’s involvement in key campus events such as Portraits of Excellence and International Student Week (ISW), where we proudly represent our heritage and foster unity among students. Overall, the VINADA Club website serves as a dynamic platform that celebrates our mission, promotes our activities, and strengthens our connection with the university community.
Key Features:
Six responsive pages: Home, About, Events, Team, Membership, and Contact.
Carousel on the homepage for visual engagement.
Message from the President section with a dynamic layout.
Executive profiles with background images.
Events preview grid and contact form with validation.
Lazy loading for images and scripts.
SEO-friendly URLs, robots.txt, and XML sitemap.
Technologies Used:
HTML5, CSS3, JavaScript (Vanilla)
Bootstrap (for layout and responsiveness)
Google Fonts (Montserrat)
Performance & Optimization:
Assets hosted on Netlify CDN for faster global delivery.
Cache-control headers and asset versioning configured.
Performance metrics improved by 20% after CDN integration, verified with Lighthouse.
Accessibility & SEO:
Semantic HTML5 structure validated with W3C.
All images include descriptive alt attributes.
ARIA roles for navigation and interactive elements.
SEO metadata and sitemap included.



CDN Provider & Configuration
CDN Used: Netlify (https://www.netlify.com)
Github link: https: https://github.com/reyneilw943-cloud/Vinada-club
Setup: Project deployed by dragging the zipped site folder onto Netlify’s interface.
Cache-Control: Netlify automatically manages caching headers for assets.
Asset Versioning: Query strings appended in HTML for CSS and JS files (e.g., style.css?v=1) to force updates.
Performance Improvement: Lighthouse reports show an increase in site loading performance by approximately 20% due to CDN caching and distribution.
Frameworks & Libraries Used
Bootstrap 5: For responsive grid layouts and components.
Google Fonts: Montserrat font family for consistent typography.
Vanilla JavaScript: To implement interactive carousel and modal features.

Accessibility and SEO Strategies
Semantic HTML5: Use of semantic tags like <header>, <main>, <section>, <article>, and <footer>.
ARIA Roles & Attributes: Navigation and landmarks annotated (e.g., role="banner", aria-label).
Alt Text: All images have descriptive alt attributes for screen readers.
SEO Metadata: Each page includes appropriate <title>, <meta name="description">, and <meta name="keywords">.
SEO-friendly URLs: Clean URLs such as about.html and events.html.
robots.txt: In the root directory to allow all search engines.
XML Sitemap: Included for search engine crawling.
